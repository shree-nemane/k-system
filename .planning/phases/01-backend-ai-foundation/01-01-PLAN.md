<task id="T03">
  <title>Implement Static API Key Security Middleware</title>
  <read_first>
    - .planning/phases/01-backend-ai-foundation/01-CONTEXT.md — D-15 (Static API key via X-API-Key header, all endpoints)
    - backend/app/core/config.py — Settings.API_KEY value
  </read_first>
  <action>
    Create `backend/app/core/security.py`:
    ```python
    from fastapi import Security, HTTPException, status
    from fastapi.security.api_key import APIKeyHeader
    from app.core.config import settings

    api_key_header = APIKeyHeader(name="X-API-Key", auto_error=False)

    async def verify_api_key(api_key: str = Security(api_key_header)) -> str:
        if api_key is None:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="X-API-Key header is missing",
            )
        if api_key != settings.API_KEY:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Invalid API key",
            )
        return api_key
    ```
  </action>
  <acceptance_criteria>

    ```python
    from contextlib import asynccontextmanager
    from fastapi import FastAPI
    from app.routers import crowd, sos, alerts, cameras

    @asynccontextmanager
    async def lifespan(app: FastAPI):
        # Startup: initialize DB connections, etc.
        print("MahaKumbh Smart Guide API — Starting up")
        yield
        # Shutdown
        print("MahaKumbh Smart Guide API — Shutting down")

    app = FastAPI(
        title="MahaKumbh 2027 Smart Guide API",
        version="0.1.0",
        description="Real-time crowd monitoring and safety coordination API",
        lifespan=lifespan,
    )

    app.include_router(crowd.router)
    app.include_router(sos.router)
    app.include_router(alerts.router)
    app.include_router(cameras.router)

    async def health_check():
        return {"status": "healthy", "service": "MahaKumbh Smart Guide API"}
    ```
