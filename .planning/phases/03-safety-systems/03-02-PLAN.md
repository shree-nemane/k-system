## Verification Criteria
### Automated
- Integration test: Mock network failure, verify pulse persists in Isar, verify WorkManager retries and eventually succeeds when network is "restored".
- Backend unit tests for status state transitions.

### Manual
- Trigger SOS, wait for countdown.
- Force kill app immediately after countdown.
- Check backend logs: verify pulse is received after 30s-1m via WorkManager.
- Verify push notification (FCM) or polling updates the UI to "Received" and "Dispatched".
